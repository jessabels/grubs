# Grubs

Not sure what to cook? Grubs will help you out with that. Grubs is a "Tinder for Recipes", meaning you can simply swipe right to save a recipe, or swipe left if it is not what you're looking for.

### Technologies Used

- Javascript
- React.js
- Node.js
- Express
- Redux
- PostgreSQL
- Sequelize
- Material-UI
- CSS
